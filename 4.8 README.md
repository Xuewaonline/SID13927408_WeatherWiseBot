# WeatherWiseBot - Intelligent Weather Assistant

An intelligent weather notification system.

## ✨ Features

- 🌤️ **Real-time Weather**: Get current weather for any city worldwide
- 📅 **5-Day Forecast**: View upcoming weather predictions
- 👔 **Outfit Suggestions**: Smart clothing recommendations based on weather
- 📱 **SMS Notifications**: Send weather information via SMS
